# פרשת נשא
בפרק ז פסוק י״ט רשי מבאר מדוע יששכר זכה להקריב מיד אחרי יהודה? משום שהיו יודעים בתורה שנאמר ומבני יששכר יודעי בינה לעיתים, ודבר נוסף שהם מצנו את העצה לנשיאים להביא קרבנות הללו. 
חשבתי לבאר על פי המהר״ל שידיעת תורה באה לידי ביטוי דווקא בתורה שבעל פה שהיא תורה שצריך לדעת אותה כי בשונה מתורה שבכתב היא לא נמצאת בארון הספרים, והיא התורה שבזכותה הקב״ה כרת ברית עם עם ישראל כפי שכתוב בירושלמי בפאה בפרק שני וגם הפסוק שרש״י מביא יודעי בינה לעיתים מדבר על הידיעה בקביעת המועדות שזה תורה שבעל פה. והדבר הנוסף שהתורה שלהם הייתה תורה שמביאה לידי עשייה ולא רק ידיעה בעלמא שהם השפיעו על כלל השבטים להביא את הקרבנות לחנוכת המזבח.
בזכות הנ״ל הח זכו להקריב מיד אחרי המלך יהודה.
